# FNP Sales Dashboard (Excel)

An interactive Excel-based Sales Analytics Dashboard built using Pivot Tables, Pivot Charts, KPIs, and slicers to analyze e-commerce sales data from FNP. This dashboard provides end-to-end insights into revenue trends, order patterns, delivery timelines, and product performance.
